## Dependencies
- `vite`
- `react`
- `react-dom`
- `framer-motion`

## Setup
1. Clone the repository
2. `cd thewalledcity`
3. Install necessary dependencies
    - Using `npm`: `npm -i`
    - Using `yarn`: `yarn install`
4. Run the dev script
    - Using `npm`: `npm run dev`
    - Using `yarn`: `yarn run dev`
